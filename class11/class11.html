# Use only the conservation scores that match the number of residues
sim.use <- sim[1:n.res]

# Map conservation scores onto the residue numbers in the PDB
occ <- vec2resno(sim.use, m1.pdb$atom$resno)

# Write a new PDB file with conservation scores in the occupancy column
write.pdb(m1.pdb,
          o = occ,
          file = "m1_conserv.pdb")

HIV protease AlphaFold model colored by residue conservation scores. Highly conserved active site residues are visible in the central region of the structure.

Summary

AlphaFold successfully predicted HIV protease structures with generally high confidence scores. RMSD and RMSF analyses showed structural similarities and variable regions among models. PAE analysis demonstrated that some models were more reliable than others. Conservation analysis identified highly conserved residues that are likely important for HIV protease function and catalytic activity.
